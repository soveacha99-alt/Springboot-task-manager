# Spring Boot Task Manager API

## Project Overview
This project is a simple Task Manager REST API built using Spring Boot.  
It allows users to create tasks, view all tasks, and delete tasks.

## Features
- Create tasks
- View all tasks
- Delete tasks

## Technologies Used
- Java
Spring Boot
Spring Data JPA
H2 Database
Gradle
GitHub

## API Endpoints
GET /tasks  
POST /tasks  
DELETE /tasks/{id}

## Run Locally
Open the project in IntelliJ IDEA
Run `HelloWorldWebAppApplication`
Use this link "http://localhost:8080/" to open spring boot in any browsers.

## Deployment
