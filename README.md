# Spring Boot Task Manager API

## Project Overview
This project is a simple Task Manager REST API built using Spring Boot.  
It allows users to create tasks, view all tasks, and delete tasks.

## Features
- Create tasks
- View all tasks
- Delete tasks

## Technologies Used
- Java
Spring Boot
Spring Data JPA
H2 Database
Gradle
GitHub

## API Endpoints
GET /tasks  
POST /tasks  
DELETE /tasks/{id}

## Run Locally
1. Open the project in IntelliJ IDEA
2. Run `HelloWorldWebAppApplication`
3. Use this link "http://localhost:8080/" to open spring boot in any browsers.

## Deployment
This project can be deployed using Railway.

Steps:
Push the project to GitHub
   Go to railway.app
Login with GitHub
Deploy the repository
