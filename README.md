# Spring Boot Task Manager API

## Project Overview
This project is a simple Task Manager REST API built using Spring Boot.  
It allows users to create tasks, view all tasks, and delete tasks.

## Features
- Create tasks
- View all tasks
- Delete tasks

## Technologies Used
- Java
Spring Boot
Spring Data JPA
H2 Database
Gradle
GitHub

## API Endpoints
GET /tasks  
POST /tasks  
DELETE /tasks/{id}

## Run Locally
1. Open the project in IntelliJ IDEA
2. Run `HelloWorldWebAppApplication`

## Deployment
This project can be deployed using Railway.

Steps:
1. Push the project to GitHub
2. Go to railway.app
3. Login with GitHub
4. Deploy the repository
